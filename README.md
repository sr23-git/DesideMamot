# マーモット決闘会議 (Deside - Marmot Battle)

迷ったとき、マーモットたちがあなたの代わりに決断を下してくれる意思決定支援ツールです。

## 概要

2つまたは3つの選択肢を入力し、「対決スタート」ボタンを押すと、マーモットたちがアリーナで激しい（もふもふした）決闘を繰り広げます。最後に勝ち残ったマーモットの掲げる候補が、今日のあなたの「決定」です。

## 特徴

- **2人戦 / 3人戦対応**: 状況に合わせて対戦人数を切り替え可能。
- **こだわり抜かれたアニメーション**: じりじりと間合いを詰める緊迫感から、もふもふの乱戦、そして劇的な勝利宣言までを CSS アニメーションで再現。
- **レトロゲーム風 SE**: Web Audio API を活用した、懐かしくも温かみのあるサウンド。
- **レスポンシブデザイン**: PC でもスマートフォンでも快適に動作します。

## 使い方

1. モードボタンで「2人戦」または「3人戦」を選択します。
2. 入力欄に決めたい候補を入力します（例：今日のランチ、週末の行き先など）。
3. 「対決スタート」をクリック！
4. マーモットたちの決闘を見守ります。
5. 勝者が決まると、画面中央に大きく決定事項が表示されます。

## 技術スタック

- **HTML5 / CSS3**: グラスモーフィズムや Container Queries を取り入れたモダンなスタイリング。
- **JavaScript (Vanilla)**: 外部ライブラリに頼らない軽量なロジック実装。
- **Web Audio API**: プロシージャルなサウンド生成。
